# Security Best Practices

Using blockchain applications requires attention to security.

## Wallet Safety

- Store your seed phrase offline.
- Never share your private keys.
- Use a strong password for your wallet.

## Transaction Verification

- Review transaction details before signing.
- Verify the network and recipient address.
- Be cautious when interacting with unfamiliar applications.

## Website Verification

- Use official project websites.
- Avoid clicking suspicious links from social media or messages.
- Bookmark frequently used services.

## Smart Contract Risks

All smart contracts carry some level of risk.

Users should research projects and understand the risks before depositing funds.
