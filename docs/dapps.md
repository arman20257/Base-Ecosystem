# dApps on Base

The Base ecosystem includes a growing number of decentralized applications.

## Categories

### Decentralized Exchanges (DEXs)

Users can swap tokens and provide liquidity through decentralized exchanges available on Base.

### DeFi Protocols

Various lending, borrowing, and yield-generating applications are being developed on Base.

### NFT Platforms

Creators and collectors can interact with NFT marketplaces that support Base.

### Social Applications

Some projects are exploring on-chain social experiences and creator tools.

## Things to Consider

- Check transaction fees before interacting with a protocol.
- Verify official project links.
- Understand the risks associated with smart contracts.
## Exploring New dApps

When trying a new decentralized application:

1. Read the project's documentation.
2. Verify the official website.
3. Start with a small transaction.
4. Review the permissions requested by your wallet.
5. Stay informed about project updates.
