# Wallets Compatible with Base

Several popular wallets support the Base network.

## MetaMask

MetaMask can be connected to Base by adding the network configuration manually or through supported applications.

## Coinbase Wallet

Coinbase Wallet provides native support for the Base ecosystem and is commonly used by new users.

## Rabby Wallet

Rabby offers a user-friendly interface for interacting with multiple EVM-compatible networks, including Base.

## Security Tips

- Never share your seed phrase.
- Verify URLs before connecting your wallet.
- Use hardware wallets for large amounts.
