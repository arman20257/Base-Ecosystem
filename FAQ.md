# Frequently Asked Questions

## What is Base?

Base is an Ethereum Layer 2 network designed to provide faster and lower-cost transactions.

## Do I need a special wallet?

No. Many Ethereum-compatible wallets can also be used with Base after adding the network.

## Is Base compatible with Ethereum?

Yes. Base is EVM-compatible, making it easier for developers and users to interact with Ethereum-based applications.

## Who is this repository for?

Anyone interested in learning the basics of the Base ecosystem.
